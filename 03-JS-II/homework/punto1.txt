* `for`: Un bucle del tipo "for" sirve para repetir un bloque de código varias veces, definiendo 
las condiciones con las cuales se va a ejecutar dicho bucle.

* `&&`, `||`, `!`:

El operador "&&" ("Y" o "AND") sirve para comparar 2 expresiones y devolver true 
solo si las dos son positivas, de lo contrario el resultado será false.

El operador "||" ("O" u "OR") sirve para comparar 2 expresiones y devolver true 
si una o ambas son positivas, de lo contrario el resultado será false.

El operador  "!" ("NO" o "NOT") devolverá el valor booleano contrario al que 
devolvería en el caso de no estar este mismo, sirve para poner la condición de 'si no' en vez de 'si' ej:
!0==='0' en este caso dice si 0 NO es igual a '0', y ahora se ejecutaría cierto código.
